Создание GUI с помощью библиотеки tkinter



Содержание страницы:



• Создание главного экрана

• Создаем первое окно



Эта библиотека называется tkinter. С помощью этой библиотеки можно создать вполне приличное меню пользователя и внешний вид программы. Ярким примером использования библиотеки tkinter для создания приложений служит окно интерпретатора Python — IDLE, или окно встроенного текстового редактора. Все они собраны с помощью этой библиотеки.


На этом уроке мы рассмотрим как создавать главное окно. В нем будут размещаться все остальные элементы — поля ввода, кнопки и др. Создание этих элементов мы рассмотрим в следующих статьях.



Создание главного экрана


Для создания главного экрана нужно импортировать библиотеку tkinter (учтите, что эта библиотека не входит в стандартный интерпретатор в Linux-системах). Импортировать библиотеку можно несколькими способами:


• import tkinter

• import tkinter as tk

• from tkinter import *


Каждый способ имеет как свои преимущества, так и недостатки. Давайте рассмотрим их.

>>>import tkinter

Этот способ, если исходить из надежности, самый правильный. Доступ к модулям и элементам этой библиотеки осуществляется через префикс tkinter.имя_элемента. Это позволяет более свободно использовать поле имен переменных — едва ли вы создадите переменную с таким именем.

>>>import tkinter as tk

Этой инструкцией мы импортируем библиотеку tkinter, но доступ к ней теперь осуществляется через префикс tk.имя_элемента. Это может быть полезно, если вы, к примеру, используете еще одну библиотеку графического интерфейса, имеющую аналогичные классы. Например библиотека ttk.

>>>from tkinter import *

Этот способ нужно использовать лишь в самых простых программах. Дело в том, что при таком способе использования, в пространство имен программы импортируются всем названия классов этой библиотеки. Теперь можно напрямую обращаться к названиям классов для создания графических элементов. Но это существенно ограничивает список доступных для использования элементов (вы не сможете использовать переменные Button, Text и пр. без ущерба для работы программы).



Создаем первое окно


Окно для программы Python создается в несколько строчек. Так как наша программа будет минимальной, используем третий способ для импорта библиотеки tkinter. Наберите в текстовом редакторе и сохраните под именем Test_gui.py.

>>>from tkinter import *

>>>root = Tk()

>>>root.mainloop()

Вы увидите новое окно с заголовком tk. Это и сеть ваше первое окно графического интерфейса. Все остальные элементы будут так или иначе находится в нем. Окно можно развернуть на весь экран, свернуть и закрыть. Всего одна строка. Просто, не правда ли?


Последняя строка, root.mainloop(), запускает цикл обновления созданного окна. Это позволяет программе отслеживать действия пользователя и выполнять соответствующие действия. Она всегда должна быть последней.


REM: Кстати, из-за того, что главное окно работает в цикле, использование в функциях цикла «while True», скорее всего, приведет к неработоспособности программы. Старайтесь использовать цикл for, если требуется выполнить какое-то количество повторяющихся действий!


Теперь давайте изменим заголовок нашего окна. Для этого измените код программы следующим образом:


>>>from tkinter import *

>>>root = Tk()

>>>root.title('Приложение')

>>>root.mainloop()

Запустите программу. Появится уже знакомое окно, но в заголовке у нее теперь отображается строка «Приложение». Следующим шагом мы изменим фоновый цвет окна. Добавьте в программу еще одну строку:

>>>from tkinter import *

>>>root = Tk()

>>>root.title('Приложение')

>>>root['bg'] = 'green'

>>>root.mainloop()

Запустите программу. Фон окна стал зеленым. Кроме прямого указания цвета (к тому же количество доступных имен цветов сильно ограничено) можно указывать нужный цвет в 16-ричном формате:

>>>from tkinter import *

>>>root = Tk()

>>>root.title('Приложение')

>>>root['bg'] = '#aa00aa'

>>>root.mainloop()

Эта программа создаст окно с фиолетовым фоном.


Кроме заголовка и цвета фона, окну можно указать размеры окна в пикселях. Наберите и запустите следующие программу:


>>>from tkinter import *

>>>root = Tk()

>>>root.title('Приложение')

>>>root['bg'] = '#aa00aa'

>>>root.geometry('400x400')

>>>root.mainloop()

После запуска вы увидите окно с фиолетовым фоном, размером 400×400 пикселей. Так можно заранее задавать размеры окна при запуске программы. Однако, иногда бывает нужно, чтобы при запуске программы она разворачивалась во весь экран. Как этого добиться? Ведь существует несколько основных разрешений для мониторов домашних ПК — и они существенно отличаются как по количеству пикселей, так и по соотношению сторон. Чтобы добиться этого, воспользуйтесь следующими командами:

>>>from tkinter import *

>>>root = Tk()

>>>root.title('Приложение')

>>>root['bg'] = '#aa00aa'

>>>root.state('zoomed')

>>>root.mainloop()

Вновь созданное окно автоматически развернется на весь экран при запуске. Фактически, это готовое окно вашего будущего приложения. нем еще нет меню, кнопок и других элементов — все это появится по мере того, как вы будете изучать этот цикл статей.


После запуска, вы наверное заметили, что хотя окно и развернулось во весь экран, панель «Windows» все равно остается доступной. Что бы полностью занять всю доступную поверхность экрана, следует несколько изменить код программы:


>>>from tkinter import *

>>>root=Tk()

>>>root['bg'] = '#aa00aa'

>>>root.overrideredirect(1)

>>>root.state('zoomed')

>>>root.mainloop()

Перед запуском учтите, что окно заполнит собой весе пространство экрана- не будет видно ни панели «Windows», ни кнопок закрытия окна. Не волнуйтесь, окно не заблокировало систему — нажмите комбинацию клавиш «Arl+Tab» и переключитесь в Windows. Закройте окно интерпретатора IDLE и окно tkinter пропадет тоже.
